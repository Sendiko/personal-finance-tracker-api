Core Features

üë§ User Management

  • Registration: Name, Email, Password with secure hashing.
  • Authentication: Secure login and session management.
  • Profile: Users can view and update their name or change password.

üí≥ Wallet Management

  • Add/Edit/Delete: Full control over wallets (Card, E-money, Cash).
  • View Balances: A clear list of all wallets and their current balances.
  • Multi-Wallet Support: Manage several fund sources simultaneously.

üîÑ Transaction Management

  • Log Income/Expense: Quickly add transactions with details like amount, category, and wallet.
  • View & Filter: Comprehensive transaction history with powerful filtering (date, type, category).
  • Modify: Edit or delete past transactions as needed.

üè∑Ô∏è Category Management

  • Custom Categories: Create, rename, or delete categories to fit personal spending habits.
  • Default Set: Comes with a pre-populated list of common categories.
  • User-Specific: Categories are tied to the individual user account.
